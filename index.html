Proyectos

Tienda de Café Online

Un sitio web de comercio electrónico ficticio diseñado para una tienda de café. El objetivo era crear una experiencia de compra fluida con un diseño responsivo. Construido con HTML y CSS para la estructura y el estilo, y JavaScript para la interactividad, como el carrito de compras.

Mi blog de viajes

Un blog personal para documentar mis viajes y experiencias. Este proyecto me ayudó a practicar mis habilidades de diseño (CSS) y a crear un sistema de navegación sencillo e intuitivo (HTML). El diseño es minimalista y se enfoca en la legibilidad del contenido.

Blog de Comida Saludable

Mi blog personal sobre comida saludable, donde publico recetas fáciles y consejos para una vida más balanceada. Fue una oportunidad para poner en práctica mis habilidades de desarrollo front-end, creando una plataforma donde la navegación fuera intuitiva y el contenido se viera atractivo para los lectores.
